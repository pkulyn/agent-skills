# 文体模板2：工作总结

> 适用范围：机关年度工作总结、专项工作总结、阶段性工作总结
> 模型基础：三层精简结构变体
> 参考样本：《机关年度工作总结范文十篇》

---

## 一、整体结构

```
开头引言（3-5%）
├── 时段界定+总体判断

第一部分：主要工作及成绩（50-55%）
├── 一、[核心工作1]（15-20%）
│   ├── 做法与举措
│   └── 成效与数据
├── 二、[核心工作2]（15-20%）
│   ├── 做法与举措
│   └── 成效与数据
├── 三、[核心工作3]（10-15%）
│   ├── 做法与举措
│   └── 成效与数据
└── [四、其他工作]（5-10%）

第二部分：经验体会/存在问题（20-25%）
├── 二、经验体会（10-15%）
│   ├── 体会1
│   ├── 体会2
│   └── 体会3
└── 三、存在问题（10%）
    ├── 问题1
    ├── 问题2
    └── 问题3

第三部分：下一步打算（15-20%）
├── 四、下一步工作打算
│   ├── 重点工作1
│   ├── 重点工作2
│   └── 重点工作3
└── [保障措施]
```

**与政府工作报告的关键区别**：
- 无独立的"总体要求"部分
- "经验体会"独立成节（政府工作报告通常不单独列）
- "下一步打算"代替"工作任务"（更简洁）
- 篇幅重心在回顾部分

---

## 二、各部分写作要点

### 2.1 开头引言

**标准格式**：
```
[年份/时段]以来，在[上级]的正确领导下，在[相关部门]的支持配合下，
[本单位]紧紧围绕[中心工作/目标任务]，
[总体思路概括]，
较好地完成了各项工作任务。
现将[时段]工作总结如下：
```

**变体**：
- 年度总结："[X]年，我局/委/办..."
- 专项总结："自[X]月启动以来，[项目/活动]..."
- 阶段总结："[X]期间，[单位]..."

**写作要点**：
- 1-2段，交代背景+总体评价
- 总体评价要留有余地："较好完成"优于"圆满完成"

### 2.2 第一部分：主要工作及成绩

**核心原则**：成绩写够、数据支撑、亮点突出

#### 结构选择

**方式1：按工作领域分**（最常见）
```
一、[领域1]工作成效显著
二、[领域2]工作扎实推进
三、[领域3]工作取得突破
```

**方式2：按工作方法/思路分**（突出做法）
```
一、坚持[X]导向，[成效]
二、突出[X]重点，[成效]
三、强化[X]保障，[成效]
```

**方式3：按时间阶段分**（阶段性工作）
```
一、第一阶段：[起止时间]，[主要工作]
二、第二阶段：[起止时间]，[主要工作]
三、第三阶段：[起止时间]，[主要工作]
```

#### 每个核心工作的展开模板

```
一、[工作领域]：[亮点概括]。

（一）[做法/举措1]：[具体内容+数据]
（二）[做法/举措2]：[具体内容+数据]
（三）[做法/举措3]：[具体内容+数据]

[成效概括段，含关键数据]
```

**标题命名技巧**：
- "成效显著""扎实推进""取得突破""稳步提升"
- 标题包含"做法+效果"双重信息
- 避免"关于XX工作的总结"（太泛）

**数据使用规范**：
```
❌ 完成了大量工作
✅ 完成[X]项任务，同比增长[X]%

❌ 取得较好效果
✅ [指标]提升至[X]，较上年提高[X]个百分点

❌ 覆盖面广
✅ 覆盖[X]个地区/单位，惠及[X]万人
```

### 2.3 第二部分：经验体会

**三种写法**：

#### 写法1：体会式（最常见）
```
回顾[时段]工作，我们有以下几点深刻体会：
一是必须坚持[X]。[阐述]。
二是必须坚持[X]。[阐述]。
三是必须坚持[X]。[阐述]。
```

#### 写法2：启示式
```
[时段]工作给我们的启示是：
[X]是做好工作的根本保证。
[X]是推动发展的关键所在。
[X]是取得成效的重要前提。
```

#### 写法3：做法提炼式
```
在工作中，我们形成了以下有效做法：
（一）[做法1]：[概述]
（二）[做法2]：[概述]
（三）[做法3]：[概述]
```

**写作要点**：
- 3-5条体会，每条2-3句阐述
- 体会要有思想高度，不能简单重复做法
- 用"必须""始终""坚持"等确定性语言

### 2.4 存在问题

**标准格式**：
```
在看到成绩的同时，我们也清醒认识到，
工作中还存在一些不足和问题：
一是[问题1]。[具体表现]。
二是[问题2]。[具体表现]。
三是[问题3]。[具体表现]。
```

**问题类型**：
1. **能力不足型**："[方面]能力有待提升"
2. **进展不均衡型**："[方面]进展不平衡"
3. **机制不完善型**："[方面]机制尚需健全"
4. **覆盖不到位型**："[方面]覆盖不够全面"

**写作要点**：
- 3-5个问题，每个1-2句
- 不回避、不夸大
- 先主观后客观，先内部后外部
- 不写无法解决的问题

### 2.5 第三部分：下一步打算

**标准格式**：
```
[下一步/明年]，我们将[总体思路]，
重点做好以下工作：
（一）[重点1]：[具体举措]
（二）[重点2]：[具体举措]
（三）[重点3]：[具体举措]
[保障措施/表态]
```

**写作要点**：
- 3-5项重点工作
- 每项工作要有具体抓手，不能只写方向
- 与存在问题呼应（问题→对策）
- 篇幅不宜过长（占全文15-20%）

## 四、常见问题与避坑

| 问题 | 表现 | 解决方案 |
|------|------|----------|
| 记流水账 | 事无巨细都写 | 只写亮点，一般工作一笔带过 |
| 评价失度 | 夸大成绩 | 用数据说话，留有余地 |
| 问题蜻蜓点水 | "有待提升""尚需加强" | 写具体表现和原因 |
| 体会空洞 | "领导重视""齐心协力" | 提炼到方法论高度 |
| 打算太虚 | "进一步加强""继续推进" | 写具体抓手和时间节点 |

## 五、自检清单

- [ ] 开头是否交代清楚时段、背景、总体评价
- [ ] 主要成绩是否有数据支撑（至少3处定量描述）
- [ ] 领域/板块之间是否逻辑有序、重点突出
- [ ] 经验体会是否有思想高度（不是简单重复做法）
- [ ] 存在问题是否直面且具体
- [ ] 下步打算是否与问题呼应
- [ ] 总体基调是否客观务实
- [ ] 关键表述是否与上级精神一致
