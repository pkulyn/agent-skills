# 文体模板5：调研报告

> 适用范围：专题调研报告、课题研究报告、政策建议报告
> 模型基础：四段式结构
> 参考样本：经验文章中的调研报告撰写指导

---

## 一、整体结构

```
标题 + 摘要（5%）

一、调研背景与基本情况（10-15%）
├── （一）调研背景/目的
├── （二）调研方法/范围
└── （三）基本现状概述

二、主要发现/问题分析（35-40%）← 核心部分
├── （一）发现/问题1
│   ├── 现象描述
│   └── 原因分析
├── （二）发现/问题2
│   ├── 现象描述
│   └── 原因分析
└── （三）发现/问题3
│   ├── 现象描述
│   └── 原因分析

三、对策建议（35-40%）← 核心部分
├── （一）建议1：[具体措施]
├── （二）建议2：[具体措施]
├── （三）建议3：[具体措施]
└── （四）建议4：[具体措施]

四、结语（3-5%）
├── 总结+展望

附件（可选）
├── 数据附表
├── 调研问卷
└── 典型案例
```

**与其他文体的关键区别**：
- 调研报告重**分析**（为什么），不重**成绩**（做了什么）
- 建议部分占比最大，且必须**可操作**
- 必须有**调研方法**说明（保证可信度）
- 可以有**不同观点**的呈现

---

## 二、各部分写作要点

### 2.1 标题

**标准格式**：
```
关于[调研主题]的调研报告
```

**优化格式**（更有吸引力）：
```
[核心发现/判断]——关于[主题]的调研报告
```

范例：
- ❌ 关于我市基层治理的调研报告
- ✅ "最后一公里"仍需打通——关于我市基层治理的调研报告
- ✅ 破解"老大难"需下"硬功夫"——关于XX问题的调研报告

### 2.2 摘要（可选但推荐）

**格式**：
```
[调研时间]，[单位]组织开展了[主题]调研，
[调研范围/方法]。
调研发现：[核心发现1-3条]。
建议：[核心建议1-3条]。
```

**写作要点**：
- 200-500字
- 让读者5分钟掌握核心内容
- 写完全文后再写摘要

### 2.3 第一部分：调研背景与基本情况

#### （一）调研背景

**标准格式**：
```
为[目的]，根据[依据]，
[单位]于[时间]组织开展了[主题]专题调研。
```

**展开方式**：
```
[大背景1-2句]→[具体问题1句]→[调研必要性1句]
```

#### （二）调研方法

**必须包含**：
```
调研时间：[起止时间]
调研范围：[覆盖地区/单位/人群]
调研方法：[座谈/问卷/走访/数据采集等]
样本情况：[座谈会X场、问卷X份、走访X处]
```

#### （三）基本现状

**概括式**（1-2段）：
```
目前，[总体情况判断]。
主要表现在：
一是[现状1]；
二是[现状2]；
三是[现状3]。
```

### 2.4 第二部分：主要发现/问题分析

**核心原则**：调研报告的灵魂在"发现"，不在"罗列"

**三种组织方式**：

#### 方式1：按问题分类（最常见）
```
二、存在的主要问题

（一）[问题1]：[现象+数据+案例]
    原因分析：[深层次原因1-2条]

（二）[问题2]：[现象+数据+案例]
    原因分析：[深层次原因1-2条]

（三）[问题3]：[现象+数据+案例]
    原因分析：[深层次原因1-2条]
```

#### 方式2：按发现分类（有新发现时）
```
二、调研主要发现

（一）[发现1]：出乎意料的是...
（二）[发现2]：值得关注的是...
（三）[发现3]：令人担忧的是...
```

#### 方式3：按层次递进（深入分析时）
```
二、问题分析

（一）表层问题：[直接表现]
（二）深层矛盾：[结构性/制度性原因]
（三）根本症结：[核心矛盾]
```

**每个问题/发现的展开规范**：
```
[标题1句概括]

现象描述：[2-3句，含数据/案例]
数据佐证：[关键数据]
典型案例：[1个具体案例]（可选）
原因分析：[1-2条深层原因]
影响判断：[1句影响评估]
```

**原因分析的常用框架**：
- 体制机制：制度不完善、权责不清晰
- 资源保障：投入不足、人才匮乏
- 思想认识：重视不够、观念落后
- 方法手段：方式陈旧、手段单一
- 外部环境：政策变化、竞争加剧

### 2.5 第三部分：对策建议

**核心原则**：建议要"准、实、新"

**准**：针对问题，不偏不倚
**实**：有具体抓手，可落地
**新**：有新思路，不是老调重弹

**建议的三个层次**：

```
近期可做（立行立改）→ 中期推进（制度完善）→ 远期规划（深化改革）
```

**每条建议的展开规范**：
```
（X）[建议标题]。

[1句核心建议]。

具体措施：
1. [措施1]：[具体内容]
2. [措施2]：[具体内容]
3. [措施3]：[具体内容]

预期效果：[1句]
```

**建议标题命名技巧**：
```
✅ "完善[X]机制，解决[X]问题"     → 机制+目标
✅ "以[X]为突破口，推动[X]"       → 突破口+目标
✅ "创新[X]模式，提升[X]水平"     → 创新+提升
✅ "坚持[X]导向，优化[X]布局"     → 导向+优化

❌ "加强[X]"                      → 太笼统
❌ "进一步[X]"                    → 太虚
❌ "争取上级支持"                  → 太依赖
```

**建议与问题的对应关系**：
```
问题1→建议1（直接对应）
问题2→建议2（直接对应）
问题3→建议3+4（一个问题可能需要多条建议）
[额外建议]（调研中发现的新思路）
```

### 2.6 第四部分：结语

**简洁版**：
```
以上建议仅供参考。不当之处，请批评指正。
```

**完整版**：
```
[主题]事关[重要性判断]。
解决[核心问题]既需要[短期举措]，
更需要[长期坚持]。
我们相信，在[上级部署]下，
通过[关键举措]，
[正面展望1句]。
```

## 三、调研报告的质量标准

### 3.1 四个"真"

1. **真调研**：有实地走访、有原始数据、有第一手资料
2. **真问题**：不是想象中的问题，而是调研中发现的问题
3. **真分析**：不是浅层描述，而是深入剖析原因
4. **真建议**：不是放之四海皆准的建议，而是具体可操作的

### 3.2 数据使用规范

```
❌ 据了解大部分...（模糊来源）
✅ 问卷调查显示，78.3%的受访者...（精确数据+来源）

❌ 很多单位反映...（无具体数量）
✅ 走访的20家单位中，15家反映...（有范围有比例）

❌ 存在严重问题（主观判断）
✅ 在[指标]上，[对比数据]差距明显（客观呈现）
```

### 3.3 案例使用规范

- 每个重要发现配1个典型案例
- 案例要真实、具体、有代表性
- 隐去敏感信息（人名用XX代替）
- 案例放在现象描述之后

---

## 四、常见问题与避坑

| 问题 | 表现 | 解决方案 |
|------|------|----------|
| 调研不实 | 没有实地调研数据 | 必须有第一手资料 |
| 问题空泛 | "存在问题""有待提高" | 写具体现象和数据 |
| 分析不深 | 只描述现象不分析原因 | 每个问题必问"为什么" |
| 建议太虚 | "加强领导""提高认识" | 写具体措施和实施路径 |
| 建议与问题脱节 | 建议不对应问题 | 严格一一对应 |
| 照搬政策文件 | 大段引用上级文件 | 立足调研发现提建议 |

## 五、自检清单

- [ ] 是否说明了调研方法、范围和样本
- [ ] 每个发现是否有数据/案例支撑
- [ ] 问题分析是否触及深层原因
- [ ] 建议是否与问题一一对应
- [ ] 建议是否具体可操作（有实施路径）
- [ ] 数据来源是否可追溯
- [ ] 是否有不同观点的呈现
- [ ] 全文逻辑是否"发现→分析→建议"贯通
