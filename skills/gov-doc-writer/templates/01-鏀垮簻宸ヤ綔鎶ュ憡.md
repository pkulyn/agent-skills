# 文体模板1：政府工作报告

> 适用范围：各级政府年度工作报告、部门年度工作报告
> 模型基础：三层精简结构
> 参考样本：2022-2026年国务院政府工作报告

---

## 一、整体结构

```
开头引言（3-5%）
├── 代表/同志们问候
├── 报告性质说明
└── 请予审议/审查

第一部分：工作回顾（25-35%）
├── 一、[上年]工作回顾
│   ├── （一）总体评价
│   ├── （二）关键数据
│   ├── （三）分领域进展
│   │   ├── 领域1进展
│   │   ├── 领域2进展
│   │   └── ...（6-10个领域）
│   ├── （四）存在问题
│   └── （五）[经验体会]

第二部分：总体要求（10-15%）
├── 二、[当年]经济社会发展总体要求和政策取向
│   ├── （一）指导思想
│   ├── （二）主要预期目标
│   └── （三）宏观政策取向

第三部分：工作任务（50-60%）
├── 三、[当年]政府工作任务
│   ├── （一）领域1：定位+举措
│   ├── （二）领域2：定位+举措
│   └── ...（9-10个领域）

结尾（2-3%）
├── 号召+展望
└── 致谢
```

---

## 二、各部分写作要点

### 2.1 开头引言

**标准格式**：
```
各位代表/同志们：
现在，我代表[机构]，向大会作政府工作报告，请予审议/审查，
并请[列席人员]提出意见。
```

**要点**：
- 简洁，不超过3句
- 明确报告人、报告性质、审议要求

### 2.2 第一部分：工作回顾

#### （一）总体评价（1-2段）

**句式模板**：
```
过去一年/五年，在[上级]坚强领导下，我们[总体评价]。
面对[重大挑战/形势]，我们[应对概括]，[总体成效判断]。
```

**范例**（2024年报告）：
> 过去一年，是全面贯彻党的二十大精神的开局之年，本届政府依法履职的第一年。面对异常复杂的国际环境和艰巨繁重的改革发展稳定任务...

**写作要点**：
- 定调：用1句话概括全年工作的"底色"
- 形势：点明面对的挑战/机遇
- 总判断：1个核心判断词（"圆满完成""来之不易""显著成效"等）

#### （二）关键数据（1-2段）

**必须包含的核心指标**：
```
国内生产总值增长[X]%
城镇新增就业[X]万人
城镇调查失业率[X]%
居民消费价格上涨[X]%
粮食产量[X]万亿斤
...
```

**写作要点**：
- 数据按重要性排列
- 关键数据后可加"好于预期""完成全年目标"等判断
- 对比数据用"提高/下降X个百分点"

#### （三）分领域进展（6-10个领域，每个领域2-4段）

**领域标题句式**：
```
[动词]+[领域名词]，[要求/效果]。
```

常用动词排序：着力>推进>深入>扩大>加强>加快>坚持>实施>做好

**每个领域展开方式**：
```
[领域定位1-2句]。[总体进展概括]。
[具体进展1]：[数据/事实]
[具体进展2]：[数据/事实]
[具体进展3]：[数据/事实]（如有）
```

**领域排列顺序**（从2022-2026年报告提炼）：
```
1. 宏观调控/国内市场     ← 经济基础
2. 产业体系/新动能       ← 发展动力
3. 科教/科技自立自强     ← 创新驱动
4. 改革                 ← 体制活力
5. 对外开放             ← 外部环境
6. 乡村振兴/三农        ← 基础支撑
7. 区域协调/城镇化      ← 空间布局
8. 民生保障             ← 人民福祉
9. 绿色转型/生态文明    ← 可持续发展
10. 安全/风险防范       ← 底线保障
```

#### （四）存在问题（1-2段）

**句式模板**：
```
在肯定成绩的同时，我们也清醒看到面临的困难和挑战。
[问题1]。[问题2]。[问题3]。[问题4]。
[应对态度1句]。
```

**写作要点**：
- 3-5个问题，每个1-2句
- 不回避、不夸大
- 先宏观（经济层面）后微观（具体领域）
- 最后1句表态"我们一定高度重视，认真加以解决"

### 2.3 第二部分：总体要求

#### （一）指导思想（1-2段）

**标准格式**：
```
今年[工作/发展]的总体要求是：
以习近平新时代中国特色社会主义思想为指导，
全面贯彻落实党的二十大和二十届二中、三中全会精神，
按照[上级会议/文件]部署，
坚持[总体原则]，
统筹[重大关系]，
[核心任务概括]。
```

**写作要点**：
- 严格遵循上级最新表述
- 不创新、不省略关键提法
- 每次换届/重大会议后更新

#### （二）主要预期目标（1段）

**格式**：
```
今年发展主要预期目标是：
国内生产总值增长[X]%左右；
城镇新增就业[X]万人以上；
城镇调查失业率[X]%左右；
居民消费价格涨幅[X]%左右；
居民收入增长和经济增长基本同步；
国际收支保持基本平衡；
粮食产量[X]万亿斤以上；
单位国内生产总值能耗降低[X]%左右；
...
```

**写作要点**：
- 目标数量通常6-8个
- 增长类用"左右"，底线类用"以上"
- 与上年对比调整

#### （三）宏观政策取向（1-2段）

**常见政策类型**：
- 积极的财政政策
- 稳健的货币政策
- 就业优先政策

**每个政策的展开**：
```
[政策类型]。[力度判断]。[具体取向1-3条]。
```

### 2.4 第三部分：工作任务

**每个领域的标准展开模板**：

```
（X）[动词]+[领域标题]。[定位/重要性1-2句]。

[总体要求/原则]。

[举措1]。[具体内容]。
[举措2]。[具体内容]。
[举措3]。[具体内容]。
```

**标题命名公式**：
```
着力/大力推进/深入实施/坚定不移/扩大/加强 + [核心领域] + ，+ [要求/目标]
```

**各领域内容要点**（按2024-2026年最新排列）：

| 序号 | 领域 | 核心举措数 | 典型关键词 |
|------|------|-----------|-----------|
| 1 | 国内市场/宏观 | 3-4 | 扩大内需、消费、投资 |
| 2 | 新动能/产业 | 3-4 | 新质生产力、产业体系 |
| 3 | 科技自立自强 | 2-3 | 科技创新、教育、人才 |
| 4 | 重点领域改革 | 2-3 | 市场经济、营商环境 |
| 5 | 对外开放 | 2-3 | 制度型开放、外贸外资 |
| 6 | 乡村振兴 | 2-3 | 粮食安全、乡村发展 |
| 7 | 城乡区域协调 | 2-3 | 新型城镇化、区域战略 |
| 8 | 民生保障 | 3-4 | 就业、社保、教育医疗 |
| 9 | 绿色转型 | 2-3 | 双碳、生态保护 |
| 10 | 安全风险 | 2-3 | 金融风险、安全生产 |

### 2.5 结尾

**标准格式**：
```
各位代表/同志们！
[形势判断1句]。[使命号召1句]。
让我们更加紧密地团结在[核心表述]周围，
[行动号召]，
[愿景展望]，
为[宏伟目标]而努力奋斗！
```

## 三、常见问题与避坑

| 问题 | 表现 | 解决方案 |
|------|------|----------|
| 回顾变流水账 | 面面俱到、重点不突出 | 每领域只写2-3个亮点 |
| 数据堆砌 | 只列数字不论意义 | 数据+判断，"增长X%，好于预期" |
| 部署太虚 | "要进一步""要不断"无实质内容 | 每项举措要有具体抓手 |
| 前后矛盾 | 回顾说好，问题又说不行 | 回顾讲成绩，问题讲挑战，不矛盾 |
| 领域遗漏 | 与上级部署不对应 | 对标上级报告确保领域覆盖 |

---

## 四、自检清单

- [ ] 核心经济指标是否齐全（GDP、就业、CPI等）
- [ ] 每个领域是否有2个以上数据支撑
- [ ] 问题部分是否直面而不夸大
- [ ] 目标指标是否量化、可考核
- [ ] 部署举措是否有具体抓手
- [ ] 领域排列是否遵循"经济→民生→安全"顺序
- [ ] 关键表述是否与上级最新提法一致
- [ ] 开头结尾是否呼应
